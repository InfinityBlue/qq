QQ互联SDK非官方版本
================
    
###文档结构要求:
    各语言应按照语言名称建立相关文件夹,至少要包含以下:SDK和Demo文件夹各一个,readme.md文件一个.其中,SDK内是开发人员下载的库文件,Demo内是完整的示例,readme是说明文件.以ASP语言为例:

    /                (根目录)
    /asp             (asp语言根目录)
    /asp/sdk         (asp语言的支持库)
    /asp/demo        (asp语言的示例)
    /asp/readme.md   (asp语言的说明文件)
    /asp/*           (你想放的其他文件或者文件夹)

###工作流程:
    fork项目(首次)/更新代码库 ----  修改文件  ----  提交修改  ----  发送Pull Request  ----  等待合并
